Esercitazione con Chrome DevTools

Risolvi i problemi JavaScript utilizzando gli strumenti per sviluppatori

Istruzioni per l'esercitazione

Utilizza Chrome DevTools per trovare e risolvere i problemi in questa pagina:

  • Apri Chrome DevTools (F12 o Ctrl+Shift+I)
  • Esamina la Console per vedere gli errori
  • Utilizza il pannello Sources per impostare breakpoint
  • Analizza gli elementi HTML e CSS con il pannello Elements
  • Controlla le chiamate di rete con il pannello Network
